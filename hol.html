Je vais créer une miniature de votre service 5euros ou produit pour 5 €

Je vous produis une miniature de qualité pour 5€ !

La miniature est au format : 1920 par 1080 ( px )
Format : Jpeg ( .jpg )

La miniature d'un produit est essentielle pour assurer son succès,
J'ai appris par mon expérience que tous comme dans la nature chaque
être est attirée par ce qui marque le plus son regard , c'est à dire
par ce qui a des formes visuelles parfaites , des tonalités parfaites
, des couleurs vives ( sans exagérations ) et surtout par l'originalité
et l'émotion qu'elle peut vous apporter , ces stratégies issues de
certains domaines du marketing sont implantées dans chacune de mes
miniatures afin que vous puissiez en tirer au mieux profit.

Modification :
La miniature a un détail imparfait ou une couleur / fond qui ne vous
plaît pas ? je vous propose jusqu'à trois modifications ( outre options )
Les modifications que je peux apporter :

- Luminosité
- Contraste
- Saturation
- Fond ( arrière-plan )
- Couleur générale ( exemple : bleu,vert... )
- détail perturbant
- Texte

Attention : pour un crédit d'une modification je peux ajuster plusieurs
de ces paramètres simultanément.

Pour éviter certaines erreurs/malentendus de conception,Voici comment me communiquer
comment créer votre miniature :

!! - Voir La 2nd image pour les références concernants Text1,Text2... - !!
!! - inclure les [1:' '] / [2:' '] ... - !!
!! - assurez vous de ne pas avoir insérer trop de caractères - !!

[1:'Insérer le Text1(texte en haut a gauche)<<max 19 caractères']
[2:'Insérer le Text2(texte dans le rectangle blanc aux coins arrondis)<<max 16 caractères']
[3:'Insérer le Text3(texte en dessous du précédent)<<max 19 caractères']
[4:'Insérer le Text4(description en dessous du texte précédent)<<max 145 ( 75 conseillés ) caractères']
[5:'Insérer le Text5(texte présent sur le mobile à droite)<<max 60 caractères']
[+:'couleur(exemple:bleu,vert , pas de code rgb / html ... )']

exemple de commande :

[1:'Elegant']
[2:'professionnel']
[3:'propre']
[4:'Personnalisable au gré de vos envies de nombreux thèmes']
[5:'Votre Texte plus Eclatant que jamais']
[6:'bleu']

Si vous avez choisis de commander plusieurs miniatures (options)
insérez #1ere miniature,#2eme miniature ou un séparateur du même style ( ex : A:,B: ... ).
